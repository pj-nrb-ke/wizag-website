# Prompt queue — ready to paste

Working checklist. Every prompt below is in a plain code block so it copies
cleanly into ChatGPT. Generate **one at a time**, not as a grid.

House rules are in `docs/photography-direction.md`. The short version:
bright warm daylight, vivid grade, faces engaged with the work and **never**
with the camera, no readable screens, no logos or wall slogans anywhere.

**Filenames matter.** Save each with the exact name given — replacing an
existing image is then just dropping the file over the old one.

**Before sending me a batch, check:** fingers on every hand, teeth on every
smile, and every background surface for banners, pull-up stands or logos.
