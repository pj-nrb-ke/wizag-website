# WIZAG Website

## About this project

This repository holds the source code and assets for the public website of **Wise & Agile Solutions Ltd** (WIZAG).

The goal of this project is to design and build a professional online presence that introduces the company, explains what it does, and makes it easy for visitors to learn about its services and get in touch.

## Company

**Wise & Agile Solutions Ltd** provides business solutions delivered with a practical, forward-looking approach—combining sound judgment with the flexibility to adapt as needs change.

## What the website will include

- A clear overview of Wise & Agile Solutions Ltd and its value proposition  
- Information about services and how the company helps clients  
- Contact details and ways for visitors to reach the team  
- A modern, accessible layout suitable for desktop and mobile devices  

## Repository

This project is version-controlled with Git and published to GitHub so changes can be tracked, reviewed, and deployed safely over time.

## Status

The website is in early development. Content and design will be added and refined as the project progresses.
